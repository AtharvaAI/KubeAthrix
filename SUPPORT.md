# Support

KubeAthrix is community-supported software with no guaranteed response time.

- Usage and installation questions: GitHub Discussions, if enabled.
- Reproducible defects and feature requests: GitHub Issues.
- Vulnerabilities: follow [SECURITY.md](SECURITY.md), never a public issue.
- Cluster incidents: use your organization's Kubernetes and database on-call
  process; this project is not an emergency response service.

Before requesting help, collect the chart/app version, Kubernetes version,
sanitized events, `/health/ready` result, and relevant structured logs. Never
post credentials, bearer tokens, Secret payloads, or sensitive evidence.
